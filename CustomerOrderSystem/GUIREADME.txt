GUI APPLICATION
---------------
To run the GUI:
1. Open the project in IntelliJ IDEA
2. Right-click CosGUI.java -> Run 'CosGUI.main()'

GUI Screens:
- Main Menu        : Choose Create Account, Log On, or Exit
- Create Account   : Fill in all fields; password rules enforced on submit
- Log On           : 3 attempts max; security question shown after correct password
- Customer Menu    : Access all features after login
- Browse Catalog   : Select quantity and click Add for each product
- View Cart        : See all items, subtotal, tax, and total
- Make Order       : Choose delivery method; enter new card if payment denied
- View Orders      : See all past orders with auth numbers and totals

Note: A credit card number starting with '0' simulates a bank denial in the GUI.
